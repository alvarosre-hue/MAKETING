
# Bank Marketing EDA

Proyecto de Análisis Exploratorio de Datos (EDA) desarrollado con Streamlit.

## Contenido
- app.py
- BankMarketing.csv
- requirements.txt

## Ejecución

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tecnologías
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit
